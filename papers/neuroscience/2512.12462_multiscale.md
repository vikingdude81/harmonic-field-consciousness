# Dynamical Modeling of Nonlinear Latent Factors in Multiscale Neural Activity

**arXiv:2512.12462**  
**Integration Date**: December 2025

## Paper Summary

This paper presents a framework for modeling neural activity across multiple temporal scales using nonlinear latent dynamics. The key innovation is handling different sampling rates and frequencies simultaneously, enabling real-time decoding and robust handling of missing data.

### Core Concepts

1. **Multiscale Encoder**
   - Processes neural signals at different temporal resolutions
   - Each scale captures dynamics at different frequencies
   - Lower modes (slow dynamics) best captured at coarse scales
   - Higher modes (fast dynamics) best captured at fine scales

2. **Nonlinear Temporal Dynamics**
   - Goes beyond linear state-space models
   - Captures complex interactions between timescales
   - Enables more realistic brain state modeling

3. **Real-time Recursive Decoding**
   - Streaming analysis with minimal latency
   - Incremental updates as new data arrives
   - Target: <100ms latency for consciousness state prediction

4. **Missing Data Handling**
   - Graceful degradation with incomplete observations
   - Multiscale interpolation from available data
   - Robust to 30%+ missing data

#### 1. Multiscale Encoder

The multiscale encoder processes harmonic mode data at different temporal scales:

```python
encoder = MultiscaleEncoder(n_modes=30, scales=[1, 2, 4, 8])
encoded = encoder.encode(time_series)  # Returns data at each scale
power = encoder.compute_multiscale_power(time_series)
```

Each scale has different importance weights for different modes:
- Scale 1 (finest): Captures fast oscillations (high-frequency modes)
- Scale 8 (coarsest): Captures slow dynamics (low-frequency modes)

